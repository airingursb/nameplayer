# 姓名大乐斗

## 更新日志
### 姓名大作战 1.0
* 游戏内容：输入两个人的姓名，即可进行姓名PK。

* 有六项属性值：
生命值，攻击，防御，速度，命中，运气。
除了运气，其他5项属性每个姓名都有唯一的属性值相匹配。

* 目前只支持输入中文名。

* 有三项特殊技能：九阳神功，降龙十八掌，快速治疗。

* 具体内容，大家玩玩就知道啦~
